# 🔐 Password Generator App

## 📌 Project Description
The **Password Generator App** is a simple and secure web application built using **JavaScript, HTML, and CSS**.  
It allows users to generate strong and random passwords based on selected options.

Users can customize the password by selecting different character types such as lowercase letters, uppercase letters, numbers, and symbols. The password length can also be adjusted using a slider.

This tool helps users create secure passwords for protecting their online accounts.

---

## 🚀 Features
- Generate secure and random passwords
- Adjustable **password length**
- Option to include:
  - Lowercase letters (a-z)
  - Uppercase letters (A-Z)
  - Numbers (0-9)
  - Symbols (@, *, #, etc.)
- Easy-to-use graphical user interface (GUI)
- One-click password generation

---

## 🖥️ Technologies Used
- **HTML5** – Structure of the application  
- **CSS3** – Styling and layout  
- **JavaScript** – Password generation logic

## ⚙️ How It Works
1. Select the desired **password length** using the slider.
2. Choose the character types to include:
   - Lowercase letters
   - Uppercase letters
   - Numbers
   - Symbols
3. Click the **Generate Password** button.
4. A secure password will be displayed in the input field.
